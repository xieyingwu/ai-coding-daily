⌨️ 今日 Cursor 使用技巧 · autoinstall 双阶段自举法

来源：Cursor 官方博客「用 autoinstall 自举 Composer」（2026-05-06，Shomil, Joshua & Andrew）

核心思路：Cursor 内部用「前代模型训练下一代模型的环境配置能力」——把环境设置拆成两个阶段：阶段1（目标定设阶段）让智能体输出10条关键命令和预期输出；阶段2（环境构建阶段）独立智能体自主安装依赖、模拟缺失文件，最多重试5次，跑通全部命令才算成功。Terminal-Bench 从 47.9% 提升到 61.7%。

你可以这样用：给 Cursor 任何新项目，先让它「输出10条验证环境是否 OK 的命令」，再让独立 Agent 跑通这些命令——这样后续所有 Agent 任务都有可靠的环境基础，不再因环境报错而无效消耗上下文。

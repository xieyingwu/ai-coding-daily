Claude Code 工程师揭秘 100 万 Token 上下文使用技巧：警惕"上下文腐化"

Claude Code 团队工程师 Thariq Shihipar 发布了一篇阅读量超 200 万的百万级 Token 窗口使用指南。核心警告：大上下文容易滋生"上下文腐化"——越到后面 AI 的判断越受早期对话污染，导致输出质量下降。

他分享了三条内部最佳实践：① 用"回溯"代替"纠错"（直接删掉错误对话段，重新开始），② 主动压缩上下文（让 AI 定期总结当前状态），③ 拆分任务给子 Agent（复杂任务分而治之，每个子 Agent 保持干净上下文）。对所有 Claude Code 和 Cursor 重度用户都非常实用。

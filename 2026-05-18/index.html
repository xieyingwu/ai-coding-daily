SpaceXAI Grok Build 正式发布——对标 Claude Code，Rust 原生打造，100万Token + 20 并行子 Agent，99美元/月前 6 个月
SpaceXAI Grok Build 新工具 马斯克入局

  • 并行子 Agent：最多同时开启 20 个并行子 Agent，分别承担规划、文档检索、代码编写、代码审查等任务
  • 生态兼容：完全兼容 Claude Code 生态，原有 AGENTS.md 配置、MCP 工具均可被自动识别
  • 技术实现：Rust 语言开发，Ratatui 终端框架，界面获 Hacker News 开发者好评
定价：SuperGrok Heavy 订阅 300 美元/月，前 6 个月升级优惠仅 99 美元/月。安装命令与 Claude Code 完全一致：curl -fsSL https://x.ai/cli/install.sh | bash

Cursor 集成 Microsoft Teams——@Cursor 直接在频道委托云端 Agent 任务，AI 编程渗透企业协作工具
Cursor Microsoft Teams 企业协作 云端 Agent
Cursor 于 5 月 11 日推出 Microsoft Teams 原生集成，将 AI 编程能力直接带入企业协作工作流：
  • 在任意 Teams 频道 @Cursor 即可将任务委托给云端 Agent，或将 Cursor 的代码库信息拉取到 Teams 中
  • Cursor 自动读取完整的频道对话线程，理解背景后选择合适的代码仓库和模型执行任务
  • 技术人员不需要打开 IDE，非技术人员也能直接在协作工具中触发代码任务
  • 同期 Bugbot PR 审查新增力度设置（Default / High / Max），高强度模式可多发现 35% 缺陷
这意味着 AI 编程工具的边界已从「IDE 插件」扩展至「企业协作中间件」，Project Manager 可直接在 Teams 中发起技术任务，无需打开代码编辑器。

智谱 GLM-5.1 在 Claude Code 中运行夺开源 Coding Agent 全球第一——Artificial Analysis 全新基准发布
智谱 AI GLM-5.1 开源第一 Coding Agent
全球权威评测机构 Artificial Analysis 于 5 月 12 日发布全新「Coding Agent 基准」（Artificial Analysis Coding Agent Index），覆盖 SWE-Bench-Pro-Hard-AA、Terminal-Bench v2 和 SWE-Atlas-QnA 三大主流基准，核心亮点：
  • 闭源第一：Claude Opus 4.7（在 Cursor CLI 中运行）摘得全球总榜第一
  • 开源第一：智谱 GLM-5.1（在 Claude Code 中运行）取得开源模型全球第一
  • GLM-5.1 可在单次任务中持续自主工作长达 8 小时，完成从规划、执行到迭代优化的完整闭环
  • Artificial Analysis 强调：AI 编程实际体验 = 模型能力 + Agent harness 组合，只评测模型本身已不够客观
对国内开发者的意义：GLM-5.1 接入 Claude Code 或 Cursor，可在保持顶级 Agent 执行框架的同时大幅降低推理成本，是 BYO 模型降本最新实战验证。
